Rastreability and Traceability of Medicines

Created by Mauro Cardoso using ( Python, Flask and Ganache), Â©2022


Flask frontend application created for a healthcare supply chain, using the concepts of blockchain technology

This chain simulates the transactions between actors (Hard-coded) with their respective amounts.
The Ganache application (Rapid Ethereum distribuited application development) works as a Private Ethereum Network,
that handles the actors wallets, blocks and transactions.
These transactions create a hash for each of the data involved in the transaction.
Each transactions generates the details with all the logs acting as the ledger.

Smart contracts are built in a simple layout in the solidity language, with hard-coded details.

This network is supposed to be semi-private so the spaces below, simulate the spaces reserved for each participant,
being this number reduced there is a faster consensus and a higher transaction transfer rate and greater scalability.
If there is a change in interactions it can be easy discovered through the Ganache application, or business needs, only the admin can access and modify with ease.
There is an obstacle in this implementation as it does not have a logic of permissions applied to the actors in the network and it has no structured process applied,
acting only as a simulation of interaction between interfaces with the shell of a blockchain network. .

The management of a B2B or enterprise network like this is managed by a Administrator in Ganache application.
